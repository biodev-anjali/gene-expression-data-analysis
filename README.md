# Gene Expression Data Analysis Platform

A full-stack bioinformatics web application for analyzing and visualizing gene expression datasets through a live, database-backed web interface.

## Features
- Gene expression normalization and comparison
- Interactive visualizations and plots
- Database-backed workflows using Prisma
- End-to-end full-stack deployment
- Web-based analysis interface

## Tech Stack
- Next.js
- Prisma ORM
- PostgreSQL
- Bioinformatics Data Analysis
- Vercel

## Live Demo
https://gene-expression-data-analysis-y2go.vercel.app/analyze

## Project Overview
This project demonstrates how gene expression analysis workflows can be integrated into a production-ready web platform, combining biological data analysis with modern full-stack development practices.
